# iKhwezi Livestream App - Marketing Assets

This folder contains all marketing materials for the iKhwezi Livestream App's Play Store listing and promotional campaigns.

## 📁 Folder Structure

### 📸 screenshots/
Play Store screenshots showcasing app features (required for Play Store submission)

**Required Screenshots (at least 2, up to 8):**
- `01_home_feed.png` - Main video feed with social features
- `02_live_stream.png` - Live streaming with chat and reactions
- `03_stories.png` - Stories viewer with live moments
- `04_profile.png` - User profile with creator features
- `05_login.png` - Login/Register screens
- `06_admin_panel.png` - Admin dashboard (optional)

**Specifications:**
- Resolution: 1080 x 1920 pixels (9:16 aspect ratio)
- Format: PNG or JPEG (PNG preferred for transparency)
- Size: Max 8MB per image
- Content: Show actual app UI, no mockups
- Language: English (primary)

### 🎯 icons/
App icons in various sizes for Play Store and device usage

**Required Icons:**
- `icon_512x512.png` - Play Store main icon (512x512)
- `icon_108x108.png` - Google Play icon (108x108)
- `icon_48x48.png` - Notification icon (48x48)
- `icon_192x192.png` - PWA icon (192x192)
- `icon_adaptive_foreground.png` - Adaptive icon foreground
- `icon_adaptive_background.png` - Adaptive icon background

**Specifications:**
- Format: PNG with transparent background
- Shape: Square with rounded corners handled by Play Store
- Style: Consistent with app branding (cosmic, neon theme)

### 🏷️ banners/
Promotional banners for marketing campaigns

**Banner Types:**
- `banner_1024x500.png` - Play Store feature graphic
- `banner_social_1200x630.png` - Social media sharing
- `banner_website_1200x400.png` - Website header
- `banner_email_600x200.png` - Email campaigns

**Specifications:**
- Feature Graphic: 1024 x 500 pixels
- Include: App name, key features, vibrant colors

### 🎨 feature-graphics/
Individual feature highlight graphics

**Feature Graphics:**
- `feature_live_streaming.png` - Live streaming capability
- `feature_social_chat.png` - Real-time chat and reactions

- `feature_challenges.png` - Community challenges
- `feature_watch_parties.png` - Group viewing

**Specifications:**
- Size: 1024 x 500 pixels

### 📱 mockups/

## 🎨 Design Guidelines

### Brand Colors
- Primary: Cosmic gradients (#0D0F1A to #1A1D2E)
- Accent: Neon blue (#3B82F6) and purple (#8B5CF6)
- Text: White (#FFFFFF) with glow effects
- Background: Dark cosmic theme

### Typography
- Primary: Inter font family
- Weights: 300, 400, 500, 600, 700, 800, 900
- Style: Modern, clean, readable

### Key Messaging
- "Stream the night. Shine the signal."
- "Social livestreaming made fun"
- "Real-time community engagement"
- "Instagram + TikTok + Live streaming"
