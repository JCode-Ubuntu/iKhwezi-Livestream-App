# iKhwezi Feature Graphics - Content Guide

## Play Store Feature Graphic (1024x500)
**Primary promotional image for Play Store**

Content to include:
- App name: "iKhwezi Live Stream"
- Tagline: "Stream the night. Shine the signal."
- Key features: Live, Social, Interactive
- App icon/logo
- Cosmic background theme
- Call-to-action elements

Design elements:
- Left side: App icon and name
- Center: Feature highlights
- Right side: Device mockup or feature icons
- Background: Cosmic gradient with stars
- Colors: Neon blue and purple accents

## Social Media Banner (1200x630)
**For Facebook, LinkedIn, Instagram posts**

Content structure:
- App branding and logo
- Key value proposition
- Visual elements of features
- Download call-to-action
- Social proof elements

## Website Banner (1200x400)
**Website header and landing page**

Content to include:
- Hero section design
- App screenshots
- Feature benefits
- Download buttons
- Navigation elements

## Email Banner (600x200)
**Newsletter and promotional emails**

Content to include:
- App logo and branding
- Key message or offer
- Call-to-action button
- Minimal but impactful design

---

**Design Specifications:**
- **Colors:** Cosmic theme (#0D0F1A, #1A1D2E, #3B82F6, #8B5CF6)
- **Typography:** Inter font family, various weights
- **Style:** Modern, tech-forward, engaging
- **Elements:** Streaming icons, social interaction symbols, cosmic effects
