# iKhwezi App Icons - Specifications

## Play Store Main Icon (icon_512x512.png)
- **Size:** 512 x 512 pixels
- **Format:** PNG with transparent background
- **Usage:** Play Store listing, app store displays

Design elements:
- iKhwezi logo/branding
- Cosmic theme with stars/nebula
- Neon accent colors
- Clean, modern design
- Scalable vector elements

## Google Play Icon (icon_108x108.png)
- **Size:** 108 x 108 pixels
- **Format:** PNG with transparent background
- **Usage:** Play Store app icon

## Notification Icon (icon_48x48.png)
- **Size:** 48 x 48 pixels
- **Format:** PNG with transparent background
- **Usage:** Android notifications

## PWA Icon (icon_192x192.png)
- **Size:** 192 x 192 pixels
- **Format:** PNG with transparent background
- **Usage:** Progressive Web App icon

## Adaptive Icon Foreground (icon_adaptive_foreground.png)
- **Size:** 108 x 108 pixels (safe zone: 66 x 66 pixels)
- **Format:** PNG with transparent background
- **Usage:** Android adaptive icons (foreground layer)

## Adaptive Icon Background (icon_adaptive_background.png)
- **Size:** 108 x 108 pixels
- **Format:** PNG (can be solid color or gradient)
- **Usage:** Android adaptive icons (background layer)

---

**Design Guidelines:**
- Primary color: Cosmic dark (#0D0F1A)
- Accent colors: Neon blue (#3B82F6), purple (#8B5CF6)
- Style: Modern, cosmic, tech-forward
- Elements: Stars, signal waves, camera/streaming icons
- Avoid: Text in icon, complex details that don't scale
