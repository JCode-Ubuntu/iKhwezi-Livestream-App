# Marketing Assets Generation Script

This script helps generate marketing assets for the iKhwezi Livestream App.

## Prerequisites

1. **Design Tools:**
   - Figma/Adobe XD/Photoshop
   - Screenshot capture tools
   - Icon design software

2. **Testing Device:**
   - Android device or emulator
   - iOS device or simulator
   - Clean app installation

## Generation Steps

### Phase 1: Icon Creation

```bash
# 1. Create base icon design in vector format
# 2. Export different sizes:
#    - 512x512 for Play Store
#    - 108x108 for Google Play
#    - 48x48 for notifications
#    - 192x192 for PWA
#    - Adaptive icon layers
```

### Phase 2: Screenshot Capture

```bash
# 1. Install app on test device
# 2. Navigate to each screen
# 3. Capture screenshots:
#    - Use device screenshot function
#    - Ensure proper lighting and no UI elements blocking
#    - Capture in portrait mode
#    - Edit for branding consistency
```

### Phase 3: Banner Creation

```bash
# 1. Design feature graphic (1024x500)
# 2. Create social media banners
# 3. Design website headers
# 4. Create email marketing assets
```

### Phase 4: Feature Graphics

```bash
# 1. Create individual feature highlight images
# 2. Use consistent layout and branding
# 3. Include app screenshots or icons
# 4. Add descriptive text and CTAs
```

### Phase 5: Mockups

```bash
# 1. Use device mockup templates
# 2. Insert app screenshots
# 3. Apply professional styling
# 4. Create multiple device variations
```

## Quality Checklist

### Icons
- [ ] Transparent background
- [ ] Correct dimensions
- [ ] Sharp edges (no pixelation)
- [ ] Brand colors maintained
- [ ] Scalable design

### Screenshots
- [ ] Actual app UI (no mockups)
- [ ] Clean device state
- [ ] Proper aspect ratio (9:16)
- [ ] Readable text
- [ ] No personal information

### Banners
- [ ] Correct dimensions
- [ ] Brand consistency
- [ ] Clear messaging
- [ ] High resolution
- [ ] Web optimized
